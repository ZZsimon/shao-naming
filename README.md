# 邵氏取名

一个为姓「邵」的孩子取**三字名**的小网站。名字均出自经典，讲究平仄音律，寓意力求雅致适中——不张扬、不小气。

收录 **600 个**精选佳名：楚辞诗经 / 唐诗宋词 / 中药名 三大来源，**每个来源 200 个**（男孩 / 女孩各 100），全部标注真实出处或药性。

## 功能

- **性别选择**：男孩 / 女孩
- **名字来源**：楚辞诗经 / 唐诗宋词 / 中药名（中药名取自真实中药材，附药性与取名寓意）
- **一键取名**：随机抽取一个佳名，展示：
  - 竖排姓名（含拼音）
  - 平仄标注（青色为平、朱色为仄）
  - 出处诗句与典籍
  - 寓意解读
- **查看全部候选**：列出当前条件下的所有名字，点击任意一个查看详情
- **收藏夹**：心仪的名字可一键收藏，使用浏览器 `localStorage` 持久化保存，下次打开仍在
- **名字对比**：在收藏夹中勾选 2&ndash;4 个名字，并排对比其平仄、来源、出处与寓意，便于斟酌取舍

## 设计要点

- **平仄相谐**：姓「邵 shào」为去声（仄）。名库优先收录与之搭配后读来抑扬顿挫、朗朗上口的组合，避免三字同调的沉闷。
- **寓意适中**：刻意避开「霸/天/帝/龙」等过于张扬的字，也不取过于纤弱的字，取意温润大方、清雅有节。
- **皆有出处**：每个名字均附带《诗经》《楚辞》或唐诗宋词的原句与典籍。

## 运行

无需任何依赖，纯静态页面。任选其一：

```bash
# 方式一：直接用浏览器打开
open index.html

# 方式二：起一个本地静态服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 如何添加名字

编辑 `data.js`，向 `NAMES` 数组追加一项即可：

```js
{
  name: "清川", pinyin: ["qīng", "chuān"], tones: ["平", "平"],
  source: "tang_song",   // chuci_shijing | tang_song | zhongyao
  gender: "boy",         // boy | girl
  book: "唐·王维《归嵩山作》",
  line: "清川带长薄，车马去闲闲。",
  meaning: "胸怀澄澈如清川，从容自在。",
}
```

> 平仄判断（现代普通话简化）：一声、二声为「平」；三声、四声为「仄」。
