# 邵氏取名

一个为姓「邵」的孩子取**三字名**的小网站。名字均出自经典,讲究平仄音律,寓意力求雅致适中——不张扬、不小气。

## 三级分类

按 **性别 → 来源 → 风格** 逐级选择,再一键取名:

- **性别**:男孩 / 女孩
- **来源**:楚辞诗经 / 唐诗宋词 / 中药名(中药名取自真实中药材,附药性与取名寓意)
- **风格**(随性别变化):
  - 男孩 → 沉稳清雅 / 俊逸洒脱 / 大气磊落
  - 女孩 → 温婉柔美 / 灵秀清丽 / 端庄大方

如 **邵秉文 · 邵维桢 · 邵柏笙** 即「男孩 · 楚辞诗经 · 沉稳清雅」一路:首字沉稳、次字清雅,温润正直,多出经典。

## 功能

- **一键取名**:在当前「性别 · 来源 · 风格」分类下随机抽取一个佳名,展示:
  - 竖排姓名(含拼音)
  - 平仄标注(青色为平、朱色为仄)
  - 出处诗句与典籍
  - 寓意解读
- **查看全部候选**:列出当前分类下的所有名字,点击任意一个查看详情
- **收藏夹**:心仪的名字可一键收藏,使用浏览器 `localStorage` 持久化保存,下次打开仍在
- **名字对比**:在收藏夹中勾选 2&ndash;4 个名字,并排对比其平仄、出处与寓意,便于斟酌取舍

## 设计要点

- **平仄相谐**:姓「邵 shào」为去声(仄)。名库优先收录与之搭配后读来抑扬顿挫、朗朗上口的组合,避免三字同调的沉闷。
- **寓意适中**:刻意避开「霸/天/帝/龙」等过于张扬的字,也不取过于纤弱的字,取意温润大方、清雅有节。
- **皆有出处**:每个名字均附带典籍原句,自组之名则注明立意。

## 运行

无需任何依赖,纯静态页面。任选其一:

```bash
# 方式一：直接用浏览器打开
open index.html

# 方式二：起一个本地静态服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 如何添加名字

编辑 `data.js`,向 `NAMES` 数组追加一项即可(注意标好 `source / gender / style` 三个分类字段):

```js
{
  name: "秉文", pinyin: ["bǐng", "wén"], tones: ["仄", "平"],
  source: "chuci_shijing",  // chuci_shijing | tang_song | zhongyao
  gender: "boy",            // boy | girl
  style: "沉稳清雅",         // 男:沉稳清雅/俊逸洒脱/大气磊落  女:温婉柔美/灵秀清丽/端庄大方
  book: "《诗经·周颂·清庙》",
  line: "济济多士，秉文之德。",
  meaning: "秉持文德，温文尔雅，不骄不躁。",
}
```

> 平仄判断（现代普通话简化）：一声、二声为「平」；三声、四声为「仄」。
